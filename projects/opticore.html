How to use the full OptiCore app

  1. Open the repository and install dependencies from requirements.txt.
  2. Run python opticore_sim.py to generate simulation CSV output.
  3. Start the dashboard using python -m http.server 8082, then open http://localhost:8082.
  4. Use this page’s demo and code/documentation panels as a guided reference.
